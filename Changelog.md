# TidyNotes Changelog

## [1.2.1] - 2025-12-05
- **Improvement**: Rules list now supports a global "Manage Rules" mode for reordering and deleting rules, consistent with Rulesets.
- **Improvement**: Enhanced UI density and alignment (cleaner action separators, removed empty headers, better spacing).
- **Improvement**: Improved logic label styling ("IF"/"ELSE IF") to be less distracting.
- **Fix**: Better mobile responsiveness for action inputs.

## [1.2.0] - 2025-12-04
- **Refactor**: Replaced Ruleset Modal with an inline, collapsible list in Settings.
- **Feature**: Inline editing of ruleset names.
- **Feature**: Reorder rulesets via Up/Down buttons.
- **Improvement**: "Edit Query" button replaces the icon toggle for better clarity.
- **Improvement**: Polished UI aesthetics for cleaner inputs and alignment.

## [1.0.0] - Initial Release
- Basic plugin structure.
- Ruleset creation.
- Triggers: On Load (with scheduling), Note Changes (Move Away logic, State Tracking), Manual (Custom Commands).
- Actions: Move Note, Update Property.
- UI: Comprehensive ruleset management.
