# TidyNotes

TidyNotes is a plugin for Obsidian that helps you automatically keep your notes tidy and organized. It does this by letting you create sets of rules that run when you decide. The aim is to let you think about organization less and about writing and editing more.

> [!WARNING]
> This plugin is still in early development. Please use at your own risk.

## Documentation
- [Features](Features.md): Detailed description of triggers, rules, and actions.
- [Usage](Usage.md): Guides on how to create rulesets with examples.
- [Changelog](Changelog.md): History of changes and updates.

## Installation
This plugin can be installed using the **BRAT** plugin.
1. Install **BRAT** from the Obsidian Community Plugins.
2. Add this repository to BRAT: `jbuc/obsidian-tidynotes`.
